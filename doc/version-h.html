Detailed Description

Macro Documentation

MODBUSOVER_VERSION

ModbusOver 版本号字符串.

MODBUSOVER_VERSION_MAJOR

ModbusOver 主要版本号，该版本号变化表示 ModbusOver 的功能或API发生重大变化或达到某设定目标.

MODBUSOVER_VERSION_MICRO

ModbusOver 末端版本号，该版本号变化表示 ModbusOver 的bug修复，功能和API未发生变化.

MODBUSOVER_VERSION_MINOR

ModbusOver 次要版本号，该版本号变化表示 ModbusOver 的功能或API发生变化.
